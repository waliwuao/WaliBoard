# WaliBoard

WaliBoard 是一个基于 PyQt5 开发的桌面应用程序，创建了一个可拖动、可调整大小且支持颜色选择的无边框窗口。该窗口始终置顶，可用于多种场景，如桌面覆盖层等。

## 功能特性
- 无边框窗口，始终置顶显示
- 支持拖动窗口改变位置
- 支持从四个角落调整窗口大小
- 支持通过右键菜单选择预设颜色或使用取色器自定义颜色

## 环境要求
- Python 3.x
- PyQt5
